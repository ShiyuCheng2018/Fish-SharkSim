Instruction:
This is a fish-shark simulator. You need to create a grid whose size within 10*10, then you will be required to input the detail of type of animals (fish or shark). For creating a grid, you will need to input a number for each row and col inside of Gird, then click the create button. For adding animals, you will need to create a grid first, the function will be available after creating a valid grid, then click add button. Final, click Advance Time to simulate, you also can track your actions by reading the log. Enjoy it!
